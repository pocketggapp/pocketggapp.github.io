Changelog

v2.0 [Latest]

September 21, 2024

pocketgg has been rewritten from the ground up, featuring OAuth 2.0 login, a more refined user interface, and some long-awaited improvements.

  • Added OAuth login, replacing the manual method of retrieving access tokens
  • Searching for tournaments now yields much more accurate results
  • Replaced the preset location/countries option with the ability to load tournaments near your exact current location (Requires Location Services)
  • You can now add any video game from start.gg's database, and no longer have to manually send in a Video Game Update Request for games that are missing
  • Tapping a set/match now shows the station and stream it's being played on (if available), as well as the seeds of both players
  • Added color indicators to indicate which sets/matches are in progress
  • Added support for Swiss brackets
  • Added the ability to add a tournament directly to your calendar
  • Added the ability to filter a TO's tournaments to those that they are directly organizing, are an admin of, or are competing in
  • Added an 'Online' section to the main screen for all online tournaments
  • Many other bug fixes and improvements
